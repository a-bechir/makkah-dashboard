# لوحة معلومات إمارة منطقة مكة المكرمة

## 📊 نظرة عامة

لوحة معلومات تفاعلية وعصرية لعرض مؤشرات الأداء الرئيسية (KPI) والإحصائيات لإمارة منطقة مكة المكرمة. تتضمن بيانات لأربع جهات رئيسية مع دعم نسختين من التصميم.

---

## ✨ المميزات

### 🎨 نسختان من التصميم
1. **النسخة الرسمية** - ألوان المملكة العربية السعودية (أخضر داكن وذهبي)
2. **النسخة الداكنة** - خلفية داكنة مع ألوان نيون مضيئة ومتوهجة

### 📱 تجاوب كامل
- تصميم متجاوب 100% مع جميع الأجهزة
- دعم شاشات الجوال والتابلت وسطح المكتب

### 🕌 أربعة أقسام رئيسية
1. **إدارة الحج والعمرة**
2. **وكالة الشؤون الأمنية**
3. **وكالة الشؤون التنموية (الصحية)**
4. **وكالة الإمارة للحقوق**

### 📈 رسوم بيانية متنوعة
- مخططات خطية للاتجاهات
- مخططات دائرية للنسب
- مخططات أعمدة للمقارنات
- مخططات مساحية للبيانات الزمنية

## 🚀 كيفية الاستخدام

### التشغيل المحلي
1. افتح الملف `index.html` في أي متصفح حديث
2. لا حاجة لخادم - يعمل مباشرة من الملفات المحلية

### التبديل بين النسختين
- انقر على زر 🌙 في الشريط العلوي للتبديل
- أو استخدم اختصار لوحة المفاتيح: `Ctrl + Shift + T`
- النسخة المختارة تُحفظ تلقائياً

### التنقل بين الأقسام
- انقر على التبويبات في الشريط العلوي
- كل قسم يحتوي على مؤشرات ورسوم بيانية خاصة به

## 🔄 التخصيص

### تعديل البيانات
افتح ملف `js/data.js` وقم بتعديل قيم الكائن `dashboardData`:

```javascript
const dashboardData = {
    hajj: {
        kpi: {
            total: 45287,  // عدل هذه القيمة
            weekly: 38642,
            // ...
        }
    }
}
```

### تعديل الألوان
- **للنسخة الرسمية:** افتح `css/theme-official.css` وعدل المتغيرات في `:root`
- **للنسخة الداكنة:** افتح `css/theme-dark.css` وعدل المتغيرات في `:root[data-theme="dark"]`

### إضافة مخططات جديدة
افتح `js/charts.js` وأضف دالة جديدة على غرار الموجودة:

```javascript
function createNewChart() {
    const ctx = document.getElementById('chart-id').getContext('2d');
    // ... إعدادات Chart.js
}
```

## 📝 ملاحظات مهمة

### البيانات التجريبية
- جميع البيانات المعروضة هي **بيانات تجريبية** لأغراض العرض فقط
- لربط بيانات حقيقية، قم بتعديل ملف `js/data.js` أو ربط API

### التحديثات في الوقت الفعلي
- الكود يحتوي على نظام للتحديثات التلقائية (معطّل افتراضياً)
- لتفعيله، قم بإزالة التعليق من السطر في `js/main.js`:
```javascript
realTimeUpdates.start();
```

### الأداء
- الرسوم البيانية يتم تحديثها فقط عند تغيير النسخة
- التحريكات محسّنة لأداء سلس

---

## 🔐 الأمان

- لا يتم جمع أي بيانات شخصية
- التفضيلات تُحفظ محلياً في المتصفح فقط (localStorage)
- لا توجد طلبات لخوادم خارجية (عدا CDN للمكتبات)
